# AI агрегатор постов в телеграмме
Проект представляет собой бота в телеграмме, направленный на фильтрацию постов из разных каналов. Будет реализованы следующие возможности:
- Добавление каналов в подборку
- Фильтрация постов по промту
- Агрегация постов по определённым периодам (например агрегировать все посты в одно сообщение один раз в день)
- Задавать промт для отправки сообщений (например, задать структуру “Новость: …; Из какой страны: …“)
- RAG по отфильтрованным постам, возможность задавать вопросы по тем постам которые прошли начальный фильтр и были сохранены в базу
- Возможность создавать несколько различных подборок
